# Peluquería Ana Zapater — Web

Sitio estático para Peluquería Ana Zapater (Morales del Vino, Zamora).
Desplegado en GitHub Pages. Sin dependencias, sin build.

## Despliegue inicial

```bash
git init
git add .
git commit -m "init: web Peluquería Ana Zapater"
git remote add origin https://github.com/clavijero/ana-zapater.git
git push -u origin main
```

Activar en GitHub: Settings → Pages → Branch: main → / (root) → Save.
URL: https://clavijero.github.io/ana-zapater/

## Actualizar contenido

Editar index.html o style.css directamente y hacer git push.
No se necesita ningún build ni servidor local.
